# arondight.github.io

## About

Source of [arondight.github.io](http://arondight.github.io).

## Dependencies

1. `bash`
2. `git`
3. `npm` and `npx`

## Usage

### INIT

```sh
git clone https://github.com/Arondight/arondight.github.io.git
cd arondight.github.io
npm install
```

### Build and Serve

```sh
npm run serve
```

> Then visit [localhost:4000](http://localhost:4000) in browser.

### Push

```bash
npm run deploy
```
