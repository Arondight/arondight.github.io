# USAGE

## INIT

```shell
git clone https://github.com/Arondight/arondight.github.io.git
cd arondight.github.io
./init.sh
```

Only do this at the very beginning.

## SERVE

```shell
./serve.sh
```

Then visit [127.0.0.1:4000](http://127.0.0.1:4000) in your web browser.

## DEPLOY

```shell
./deploy.sh
```

# DEPENDENCIES

1. bash
